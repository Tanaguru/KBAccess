Get KBAccess 2.0 sources

Download KBAccess 2.0 sources : 

	mkdir KBAccess-src
	cd KBAccess-src
	git clone https://github.com/Tanaguru/KBAccess.git

Install Tanaguru libraries

Copy every files from the KBAccess-src/KBAccess/kbaccess-resources/src/main/resources/lib/ folder.
This folder contains :

  • commons-entity-1.5.2.jar
  • commons-entity-jpa-1.5.2.jar
  • tgol-api-1.5.2.jar
  • tanaguru-mvn-manaul-install.sh
And install the libraries into the local maven repository by running tanaguru-mvn-manual-install.sh: 

	cp KBAccess-src/KBAccess/kbaccess-resources/src/main/resources/lib/* .
	chmod 744 tanaguru-mvn-manual-install.sh
	./tanaguru-mvn-manual-install.sh

Build KBAccess 2.0

With command line

	cd KBAccess-src/KBAccess
	mvn clean compile install

With Netbeans

Open the maven project kbaccess and all its sub projects.
Right-click on kbaccess project and click Clean and Build.
